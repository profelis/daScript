# dasSMT — Z3 SMT solver bindings for daslang

Dynamic bindings to the [Z3](https://github.com/Z3Prover/z3) C API. No link-time dependency and no compiled C++ in the module: CMake downloads the prebuilt `libz3` shared library at configure time into `<das_root>/lib/libz3.dll` (fixed name on every platform), and the generated bindings resolve symbols at runtime via `dasbind` `[extern]` annotations.

Enable with `-DDAS_SMT_DISABLED=OFF`.

## Layout

- `bindings/` — generated by `bind_z3.das` (do not edit): `z3_func.das`, `z3_struct.das`, `z3_enum.das`, `z3_const.das`
- `daslib/smt_boost.das` — raw-API helpers: context/solver lifecycle, error polling, array-argument wrappers
- `daslib/smt_expr.das` — typed symbolic values (`SInt`/`SBool`/`SReal`/`SString`), Z3py-style solver/model API
- `daslib/smt_macro.das` — `constrain` / `smt_formula` macros and `[smt_fn]` function lifting

## The three levels

**Typed values** (`require smt/daslib/smt_expr`) — arithmetic and comparisons are plain das operators; ordinary generic functions instantiate symbolically:

```das
def chebyshev(ax, ay, bx, by) => max(abs(ax - bx), abs(ay - by))   // ordinary das

with_solver() $(var s) {
    let x = s |> int_const("x")
    let y = s |> int_const("y")
    s |> add(x > 2)
    s |> add(s_and(x < 10, x + y == 7))          // && is not overloadable — s_and, or:
    s |> add(chebyshev(x, y, 0, 0) <= 8)         // generic builds a formula with SInt
    if ((s |> check()) == SmtResult.SAT) {
        print("x = {s |> model_int(x)}\n")
    }
}
```

**`constrain` macro** (`require smt/daslib/smt_macro`) — natural logical syntax; runtime das values lift automatically; the block form declares unknowns with regular `var`:

```das
s |> constrain(x > 2 && (x < 10 || y % 2 == 0))
s |> constrain() <| $() {
    var a : float                                 // fresh unknown, float → Real sort
    var z : string                                // string → Z3 sequence
    var m : uint                                  // uint → BitVec32 (wrap-around)
    length(z) < a && a < 100.0lf
    m + 1u == 0u                                  // solvable on BV: m = 0xFFFFFFFF
}
print("a = {s |> model_real("a")}, z = {s |> model_string("z")}\n")

let formula = s |> smt_formula() <| $() {        // built on s's context, not asserted
    var q : int
    q * q < 50 && q > 3
}
```

**Quantifiers** — `smt_forall`/`smt_exists` are TRUE SMT quantifiers over fresh bound variables; `all_of`/`any_of`/`all_pairs` are bounded combinators that expand over concrete indices at formula-build time (the practical encoding for fixed small n):

```das
s |> constrain(smt_forall(s) $(q : SInt) => q * 0 == 0)            // real ∀
s |> constrain(smt_exists(s) $(q : SInt) => y == q * 2)            // real ∃: y is even
s |> constrain(all_of(n) $(i) => xs[i] >= 0 && xs[i] < 100)        // expanded conjunction
s |> constrain(all_pairs(n) $(i, j) => xs[i] != xs[j])             // all pairs i < j
print(s |> to_smtlib())                                            // the exact SMT-LIB2 Z3 sees
```

**`[smt_fn]`** — an ordinary das function (with `if`s, early returns, literal-range `for`s) gains a symbolic twin; calling it with `SInt` builds a formula, calling with `int` runs concrete:

```das
[smt_fn]
def valid_cell(x, y : int) : bool {
    if (x < 0 || x >= 24 || y < 0 || y >= 12) return false
    for (i in range(0, 7)) {
        if (x >= WALL_X[i] && x < WALL_X[i] + WALL_W[i] && y >= WALL_Y[i] && y < WALL_Y[i] + WALL_H[i]) return false
    }
    return true
}
s |> add(valid_cell(ex, ey))    // whole function, loops unrolled, becomes one formula
```

## Raw API

```das
require smt/daslib/smt_boost

with_context() $(ctx) {
    print("{Z3_eval_smtlib2_string(ctx, "(declare-const x Int)(assert (> x 2))(check-sat)")}\n")
}
```

## Regenerating bindings

Requires a `dasClangBind` build (`-DDAS_CLANG_BIND_DISABLED=OFF`, libclang SDK — see `skills/clang_bind_build.md`) and the Z3 headers (the downloaded release zip ships them; the configure step caches the path as `DAS_Z3_INCLUDE_DIR`):

```
bin/daslang modules/dasClangBind/bind/bind_z3.das -- --input <z3-include-dir>
```
