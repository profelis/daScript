Report whether the RUNNING CPU supports a hardware feature by name — cpuid truth on x86/x86_64, OSXSAVE-honest ("avx*" names are true only when the OS also saves the register state they need). Known names: "sse4.2", "fma", "f16c", "avx", "avx2", "avxvnni", "avx512f", "avx512bw", "avx512vl", "avx512vnni"; unknown names and non-x86 architectures return false (fail-closed), so code gated on cpu_supports never selects instructions the box can't execute — e.g. the dasLLAMA x64 backends and the JIT's AVX2 intrinsic emission both gate on cpu_supports("avx2").
Deliberately never constant-folded: features are a property of the box the program runs on, not the box it compiled on.
